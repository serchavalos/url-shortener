# URL Shortener

Provides a backend service for submitting aliases for URLs

## Instructions

### How to run

```sh
npm install && echo "PORT=3000\nDOMAIN=locahost" > .env && npm start
```

### How to run test

```sh
# Run all types of tests
npm test

# Or by type
npm run test:unit
npm run test:integration
npm run test:e2e
#
```
